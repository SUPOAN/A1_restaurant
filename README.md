# 學期2-3: A1 打造餐廳清單

- 此專案可提供使用者觀看餐廳清單，包含餐廳名稱、類別以及網路評分。
- 提供利用輸入關鍵字搜尋餐廳的便捷服務。
- 點選入內更提供詳細資訊如：餐廳電話、地址以及店家資訊。

其餘功能，之後會陸續更新！
